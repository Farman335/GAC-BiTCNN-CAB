****** Instructions ******

1: Training and Testing datasets are provided as .txt format. 
   
2: PsePSSM-DCT, ProtTrans-T5, UniRep, and ESM-2 features can be extracted using PsePSSM-DCT.py, ProtTrans-T5.py, UniRep.py, and ESM-2.py.

3: XGBoost-FFS.py is used to select the best feature set. 

4: The best sets can be fused and provided to GAC-BiTCNN for model training.

5: ML-classifiers.py can be used for model training of ML classifiers.  


NOTE: The required libraries and dependencies are provided in requirement.txt file.
